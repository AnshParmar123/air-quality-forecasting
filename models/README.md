# Model Artifacts

This folder stores trained model artifacts exported from [aqi.ipynb](/Users/anshparmar/Desktop/PROJECTS/AQI/aqi.ipynb).

- `lightgbm_best.joblib`: saved LightGBM regressor
- `xgb_model.joblib`: saved XGBoost regressor
- `model_metadata.joblib`: metadata describing the saved model type, model path, and feature list

If you retrain the notebook, these files may be updated with new weights and feature metadata.
